# 🌭 Pistola's Hot Dog - Cardápio Digital

Um cardápio digital moderno, responsivo e interativo desenvolvido para a lanchonete **Pistola's Hot Dog**.

O projeto encontra-se atualmente em fase de desenvolvimento e, nesta primeira versão, contempla exclusivamente o **Front-end da aplicação**, com foco em experiência do usuário, responsividade e arquitetura escalável para futuras integrações.

O objetivo é permitir que clientes realizem pedidos diretamente através de um link disponibilizado nas redes sociais, com finalização via WhatsApp.

---

> ⚠️ **Status do Projeto:** Em desenvolvimento.
>
> Atualmente apenas o Front-end está implementado. As integrações com Backend, banco de dados e painel administrativo serão desenvolvidas em versões futuras.

## ✨ Funcionalidades Implementadas

### ✅ Disponível nesta versão

- Cardápio digital responsivo.
- Exibição dinâmica dos produtos.
- Organização por categorias.
- Carrinho de compras interativo.
- Alteração de quantidade dos itens.
- Remoção de produtos do carrinho.
- Modal de personalização dos produtos.
- Checkout com captura dos dados do cliente.
- Geração automática da mensagem para WhatsApp.
- Interface totalmente responsiva.
- Navbar adaptável para dispositivos móveis.
- Toasts de feedback para o usuário.
- Loading screen.
- Animações suaves.
- Estrutura preparada para futura integração com API.

## 📈 Objetivo do Projeto

Este projeto foi desenvolvido como estudo prático de desenvolvimento Front-end, aplicando conceitos modernos de HTML, CSS e JavaScript puro, além de servir como base para futuras implementações Full Stack.

Além disso, o sistema foi projetado desde o início com foco em escalabilidade, permitindo futuras integrações com serviços externos e APIs sem necessidade de grandes refatorações.
